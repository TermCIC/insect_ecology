Single Sample Outputs
This section will also show variance (s²) and Morisita’s Index (Iδ).
Core definitions
x̄ = Σx / n
Mean density, average number of insects per plant.
s² = Σ(x − x̄)² / (n − 1)
Sample variance, how unevenly insects are distributed among plants.
Iwao indices (single sample)
x* = x̄ + (s² / x̄) − 1
Mean crowding. Interpretable as the number of neighbors experienced by an average insect.
P = x* / x̄
Patchiness index (P).
  • P = 1: random
  • P < 1: uniform
  • P > 1: aggregated (clumped)
Morisita’s Index (single sample)
Iδ = n · Σ[x(x − 1)] / [Σx (Σx − 1)]
Probability that two randomly chosen insects occur on the same plant.
  • Iδ = 1: random
  • Iδ < 1: uniform
  • Iδ > 1: clumped
Taylor’s Power Law (requires multiple samples)
log(s²) = log(a) + b · log(x̄)
Taylor’s Power Law relates variance to mean across multiple independent samples (for example, different plots, dates, or fields). Parameter b is the aggregation index:
  • b < 1: uniform
  • b ≈ 1: random
  • b > 1: aggregated
Parameter a depends on sampling scale.
Iwao’s Patchiness Regression (requires multiple samples)
x* = α + β x̄
Regression of mean crowding on mean density across multiple samples.
α (basic contagion) suggests whether the fundamental unit is an individual or a cohesive group.
β (density-contingent) shows how crowding changes as density increases.
Multiple samples input
Taylor’s Power Law and Iwao’s regression need multiple samples. Put one sample per line. Each line is a list of insect counts for plants in that sample (comma, space, or tab separated).

Example:
0,1,0,2,3
1,0,0,1,0
5,4,2,0,1

Notes:
  • Lines with mean = 0 or variance = 0 are excluded from Taylor’s log regression (because log(0) is undefined).
  • Iwao regression (x* vs x̄) can still use samples with mean = 0, but samples with mean = 0 and x* = 0 add little information.
