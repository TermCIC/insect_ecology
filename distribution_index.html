Indices of Distribution Explained

Mean Density (x̄)

x̄ = Σx / n

Average number of insects per plant.

Variance (s²)

s² = Σ(x − x̄)² / (n − 1)

Describes how unevenly insects are distributed among plants.

Mean Crowding (x*)

x* = x̄ + (s² / x̄) − 1

Represents the number of neighbors experienced by an average insect.

Iwao’s Patchiness Index (P)

P = x* / x̄
  • P = 1: Random distribution
  • P < 1: Uniform distribution
  • P > 1: Aggregated distribution

Morisita’s Index (Iδ)

Iδ = n · Σ[x(x − 1)] / [Σx (Σx − 1)]
  • Iδ = 1: Random
  • Iδ < 1: Uniform
  • Iδ > 1: Clumped

Taylor’s Power Law

log(s²) = log(a) + b · log(x̄)
  • b < 1: Uniform
  • b ≈ 1: Random
  • b > 1: Aggregated

Taylor’s model predicts variance for sampling design. Iwao’s indices explain biological aggregation mechanisms.
